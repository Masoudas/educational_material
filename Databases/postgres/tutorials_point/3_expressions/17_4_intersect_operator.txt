Like the UNION and EXCEPT operators, the PostgreSQL INTERSECT operator combines result sets of two or more SELECT statements into a single result set. The INTERSECT operator returns any rows that are available in both result sets.

The following illustration shows the final result set produced by the INTERSECT operator.

$ SELECT cols from t1 INTERSECT cols from t2
