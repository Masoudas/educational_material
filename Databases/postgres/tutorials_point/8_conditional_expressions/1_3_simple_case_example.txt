Here's the first simple example:

$ SELECT title, rating,
       CASE rating
           WHEN 'G' THEN 'General Audiences'
           WHEN 'PG' THEN 'Parental Guidance Suggested'
           WHEN 'PG-13' THEN 'Parents Strongly Cautioned'
           WHEN 'R' THEN 'Restricted'
           WHEN 'NC-17' THEN 'Adults Only'
       END rating_description
FROM film
ORDER BY title;

Again, we can use case with aggregate functions:

$ SELECT
    SUM(
        CASE rating
        WHEN 'G' THEN 1 
        ELSE 0 
        END) "General Audiences",

    SUM(
        CASE rating
        WHEN 'PG' THEN 1 
        ELSE 0 
        END) "Parental Guidance Suggested",

    SUM(
        CASE rating
        WHEN 'PG-13' THEN 1 
        ELSE 0 
        END) "Parents Strongly Cautioned",

    SUM(
        CASE rating
        WHEN 'R' THEN 1 
        ELSE 0 
        END) "Restricted",

    SUM(
        CASE rating
        WHEN 'NC-17' THEN 1 
        ELSE 0 
        END) "Adults Only"

FROM film;
