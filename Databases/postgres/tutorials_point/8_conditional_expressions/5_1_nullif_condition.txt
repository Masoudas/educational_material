Me: The null if function seems like something I won't recall, but it's actually useful when it comes to division by zero for example!

Here's the syntax:
$ NULLIF(argument_1,argument_2);

The NULLIF function returns a null value if argument_1 equals to argument_2, otherwise it returns argument_1.

See the following examples:
$ SELECT
	NULLIF (1, 1); -- return NULL

SELECT
	NULLIF (1, 0); -- return 1

SELECT
	NULLIF ('A', 'B'); -- return A
