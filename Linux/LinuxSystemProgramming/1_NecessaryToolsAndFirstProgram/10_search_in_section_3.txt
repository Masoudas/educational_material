Question that still baffles me is how does one searches the linux headers for a particular purpose?

The answer I have so far is "apropos -s 3 search_topic." For example:

$ apropos -s 3 remove
...
rmdir (3posix)       - remove a directory
...
unlink (3posix) 	- remove a directry entry relative ...
...
sem_unlink (3posix)  - remove a named semaphore
shm_unlink (3posix)  - remove a shared memory object (REALTIME)
...
remove (3)           - remove a file or directory
remove (3posix)      - remove a file
...
